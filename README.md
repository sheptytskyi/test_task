# SOCIAL NETWORK : TEST TASK
<hr>

### Stack:
- Python 3.11
- Django 4.2
- Postgres 14.6
- Docker

<hr>

### Setup
1. Create .env file and set the variables
```
SECRET_KEY=
ALLOWED_HOSTS=
DEBUG=
DB_ENGINE=
DB_NAME=
DB_USER=
DB_PASS=
DB_HOST=
DB_PORT=
```
2. Build docker compose
```python
make build
```

3. Run docker compose
```python
make run
```
